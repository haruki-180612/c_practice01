# プログラミング演習　課題集

## 概要
授業「プログラミング演習」で作成したC言語の課題をまとめたリポジトリ

## 内容

### kadai1_1

内容：
一次方程式（a × x + b = 0）の解を求めるプログラム

学んだこと：
条件分岐（if文）の基本と、入力値に応じた処理の分け方

### kadai1_2
内容：
入力した数 n をもとに、掛け算表（九九表のような表）を表示するプログラム

学んだこと：
二重ループ（for文）の使い方と、配列を使わない表形式の出力方法

### kadai2_1
内容：
10個の数値を入力し、最大値・最小値・平均値を求めるプログラム

学んだこと：
配列の基本操作と、ループ処理によるデータの集計方法

### kadai2_2
内容：
入力された名前を加工し、姓を先頭にして名をイニシャル表記に変換して出力する

学んだこと：
文字配列（文字列）の扱い方と、文字コードを使った大文字変換・文字列の整形

### kadai3_1
内容：
入力した数値配列の平均値を求め、その値を基準に各要素を0または1に変換する（二値化）プログラム

学んだこと：
関数分割（input・average・binarize）と、配列の受け渡し・データ処理の流れ

### kadai3_2
内容：
入力した文字列の長さを求め、指定した文字の出現回数とその割合を計算するプログラム

学んだこと：
文字列の走査方法と、関数を使った処理の分割（長さ取得・文字カウント）

### kadai4_1
内容：
文字列中の特定の文字を大文字に変換し、その後大文字を取り除いて新しい文字列を作成するプログラム

学んだこと：
ポインタによる文字列の走査と、読み取り・書き込みポインタを使った文字列の編集方法

### kadai4_2
内容：
学生の名前と点数を構造体で管理し、点数の高い順に上位者を表示するプログラム

学んだこと：
構造体配列の扱い方と、ポインタを使った関数によるデータ処理（最大値の更新・順位付け）
